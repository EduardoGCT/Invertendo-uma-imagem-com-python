# Invertendo-uma-imagem-com-python
Esse projeto foi feito para aprender a lidar com dados binários em arquivo com Python pela faculdade

*Objetivos do programa*

Desenvolver um programa em Python que execute as seguintes operações:


1 Carregar uma imagem simples do sistema de arquivos.
2 Converter a imagem em uma representação binária usando a biblioteca PIL (Python Imaging Library).
3 Exibir os dados binários da imagem.
4 Salvar esses dados em um arquivo binário.
5 Fazer uma cópia desse arquivo binário.
6 Manipular os dados do arquivo binário cópia (por exemplo, inverter os bytes, adicionar ruído etc.).
7 Carregar a imagem modificada a partir do arquivo binário e exibi-la para ver o efeito das manipulações.
